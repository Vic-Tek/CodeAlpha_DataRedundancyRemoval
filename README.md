# CodeAlpha Data Redundancy Removal System

## Overview

A cloud-based system that detects and prevents duplicate records from being stored in a database. The application validates incoming data, identifies redundancy, and stores only unique entries.

## Features

* Duplicate record detection
* Data validation
* Cloud database integration
* User-friendly web interface
* Record management dashboard

## Technologies Used

* Python
* Flask
* MySQL
* AWS EC2
* AWS RDS
* GitHub

## Installation

```bash
pip install -r requirements.txt
python app.py
```
